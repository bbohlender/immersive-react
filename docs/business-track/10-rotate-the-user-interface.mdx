now do:

- create the `Billboard` component which returns a `<group>` and takes optional children of type `ReactNode` as parameters
- set on the `group` rotation-order="YXZ", position={[0, 0.1, 0]}, and insert the children inside the `<group>` tag
- add a `useRef<Group>(null)` and set the ref on the `<group>`
- create a `const positionHelper = new Vector3();` which we will use every frame to calculate the position of the camera
- add the `useFrame` which takes a callback in which we check if the ref.current exists, if the ref exists, we call `state.camera.getWorldPosition(positionHelper)` to compute the position of the camera and write it to our `positionHelper`, next we execute `ref.current.lookAt(positionHelper)` to make the group look at the camera
- to prevent only make our component rotate towards to camera on the y axis, we set all axis to `0` after executing `lookAt`
